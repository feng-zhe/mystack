# 鲈钓系统

## 知识储备
### 鲈鱼习性
#### 四季模式
- 春
    - 产前
        - 由深水区逐渐向浅水的硬底的平坦区域移动
        - 雄鱼在后者当中选择并打扫出一批区域，即为“产床”
        - 此阶段部分鱼会非常激进地进食，准备产季
    - 产中
        - 雌鱼在产床上产卵，雄鱼同时排出精液使其受精
        - 之后雄鱼一直守护产床，直到小鱼长大并散开
        - 雌鱼可能会去不同的产床
        - 此阶段可目视作钓，但是鱼非常警觉，基本不开口
    - 产后
        - 一部分鲈鱼退回至产前所在的离岸深水区域，一部分继续呆在产床上，还有一部分悬浮在障碍中
        - 此阶段鲈鱼主要在恢复体力，开口意愿不强
- 夏
    - 偏好温度更加凉快、含氧量更多的地方
        - 水草区
        - 遮蔽物
        - 离岸的结构
        - 水流处
- 秋
    - 依旧回到春季的产床区域，激进地追逐饵鱼，准备过冬
- 冬
    - 退回深水区，喜好有较大深浅变化的区域，方便去浅滩捕食后退回深水区
#### 咬口分类
- 饮食口
    - 需要进食补充能量，窗口期尤为明显，但其他时间也有，送到嘴边都会咬（除特殊状态如春季产床鲈鱼和高压鱼）
- 反应口
    - 对高速移动的物体，来不及仔细观察思考而作出本能的攻击行为。强力作钓的一大理论支持
- 驱逐口
    - 通过攻击来驱赶靠近的生物，常常出现在春季守护产床的鲈鱼身上
#### 温度影响
- <10C：低活性，代谢慢，偶尔进食
- 10C-20C：活性提升，觅食筑巢
- 20C-27C：活性最佳，代谢快，进食频繁
- >27C：活性开始下降，倾向于避暑
### 装备
#### 杆
- 调性（action）
    - 一般来讲，越快的调性反映渔讯越快，适合精细钓组;而中快/中等调性适合需要缓冲或者粘住鱼的饵，如VIB、crankbait等
- 强度（power）
    - 强度越轻，所能使用的最大渔线的拉力值就越低。需要配合使用的钓组和目标鱼种。
- 长度
    - 因人（主要是身高）而异、因作钓环境而异，方便舒服就行。
#### 轮
- 分类
    - 纺车轮
        - 优点
            - 泛用
            - 易用
            - 难炸线
        - 缺点
            - 前臂易疲劳
            - 精准抛投有难度
    - 水滴轮
        - 优点
            - 前臂轻松些
            - 更容易精准操控
            - 酷
        - 缺点
            - 入门时容易炸线
            - 大风下容易炸线
            - 相对较贵
            - 微物和远投需要买专门的水滴轮
            - 价格差距在性能上体现非常明显
- 属性
    - 线杯大小
        - 同一厂家，型号越大，线杯容量越大（如3000就比2000大）
    - 线杯深浅
        - 浅线杯比一般的容线量要小
    - 速比
        - 速比越大，收线速度越快
    - 其他
        - 如紧凑机身（Compact Body），快速抛投（quick fire）等
#### 线
- 编织线/布线
    - 优点
        - 同等拉力下，线径最小
        - 延展度几乎为零，有助于刺鱼
        - 拉直状态下，感度最好
        - 易于整理
    - 缺点
        - 不耐磨
        - 松线状态下，感度很差
        - 有颜色不透明，容易让鱼警惕
        - 浮水，可能影响泳姿
- 碳线
    - 优点
        - 耐磨
        - 全透明，隐蔽性较好
        - 延展性较低
        - 松线状态下，感度较好
    - 缺点
        - 有记忆性，容易打圈，不易整理
        - 远距离抛投后，由于延展性，需要暴力刺鱼
        - 沉水，可能影响泳姿
- 尼龙线
    - 优点
        - 便宜
        - 延展性好，抗冲击
    - 缺点
        - 刺鱼头疼，最好配合三本钩
        - 大多数情况下不如用碳线
#### 钩
- 分类
- 涂层
#### 线结
- 前导结
- 线钩结
#### 选购方针
- 奥卡母剃刀（如无必要，勿增实体）
- 装备服务钓组、钓组服务目标鱼
    - i.e. 发现鱼比较高压，决定使用精细钓组，那线就用6lb或8lb的，那杆子就得配UL/L/ML/M的，有条件可以配个重量大小都合适的轮子，让杆轮重心在握把处，减少前臂压力。

## 钓前准备
### 制定作钓计划
#### 被动信息收集
- 天气预报
    - 气温
    - 光照
    - 风力
    - 风向
- 地形图
    - 深水区
    - 浅水区
    - 深浅交界处
#### 确定主要的作钓对象
- 障碍
    - 浮萍
    - 水草
    - 灌木
    - 倒树
    - 芦苇蕩
- 结构
    - 深浅交接处
        - 方便鲈鱼由深入浅捕食饵鱼
    - 陡峭的崖壁
        - 一般较深，鲈鱼可能躲在底部
    - 碎石区
        - 鲈鱼藏匿，伏击甲壳类和底栖鱼类
    - 硬底浅滩的平原
        - 鲈鱼常常在此产卵和捕食
- 水域
    - 浅滩
        - 饵鱼常聚集在浅滩
    - 主河道
    - 支叉湾
    - 背水湾
    - 洄水区
    - 深槽
- 水层
    - 水面
    - 表层
    - 中层
    - 底层
### 选择装备
- 小冯套装
### 检查装备
- 检查泄力是否适当
- 检查线结（前导结、线钩结等）强度
- 检查前导磨损情况
- 检查钩子状态
    - 钩尖是否依旧锋利
    - 钩身是否有被拉弯

## 钓时指南
### 实时信息搜集
- 结构、障碍、遮蔽物的位置
- 水温
- 水体浑浊程度 (沿用KVD的定义)
    - 清澈 （clear，2ft+ visibility）
    - 微浑 （stained，8in-1.5ft visibility）
    - 浑水 （muddy，<8ft visibility）
- 鱼对饵的反映
    - 什么情况下有反应
    - 什么情况下没有反应
- 依据实时信息调整策略
### 策略
#### 春
- 搜索至产床的迁移路径
- 刺激产床引发驱逐口
    - 大泳饵
- 作钓障碍
- 定位饵鱼
#### 夏
- 钓深钓阴
- 作钓障碍
- 定位饵鱼
#### 秋
- 大水面搜索
- 作钓障碍
- 定位饵鱼
#### 冬
- 钓深钓阳
- 作钓障碍
- 定位饵鱼
